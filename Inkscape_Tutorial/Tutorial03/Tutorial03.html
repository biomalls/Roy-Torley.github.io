Tutorial 3 – Creating Basic Shapes

Many objects in the Inkscape program can be created by starting with basic shapes and then editing them to create new shapes. In this tutorial, you will use some basic shapes to create a logo.

In this tutorial on the work area, you will learn how to:

  • use tools and commands to create basic shapes.
  • copy and combine objects to create new shapes.
  • use rulers, grids, and guides as drawing aids.
  • paint objects.
  • scale objects using the bounding box.

Prepping for this tutorial

  1. Start Inkscape.

  2. Click on File > Open and open the Tutorial03Image01.svg file in the Tutorial03 folder.

  3. If you like, choose View > Zoom Out to make the finished artwork smaller, adjust the window size, and leave it on your screen as you work. Move the artwork where you want it in the window using the <Shift>+ arrow keys. If you don't want to leave the image open, choose File > Close.

Now create the start file to begin the lesson.

  1. Choose File > New to open a new, untitled document.

  2. Choose File > Save As..., name the file Logo, and select the Tutorial03 folder in the Save In menu. Leave the default file format as Inkscape SVG (*.svg). Inkscape will automatically append the filename with .svg as it saves.

Drawing the Pencil Shape

In Inkscape, you control the thickness and color of lines that you draw by setting stroke attributes. A stroke is the paint characteristics of a line or the outline of an object. A fill is the paint characteristics of the inside of an object. The default settings will let you see the objects you draw with a white fill and a black outline.

  1. Select the Zoom tool () in the toolbox, and click in the middle of the window once or twice until you are zoomed in to about 150%. (Notice that 150%, more or less, is displayed in the bottom left corner of the window.)

  2. Choose View > Guides to turn them on. Guides automatically snap the edges of objects to nearby objects or their intersect points as you move them.

How Guides Work

When Guides are turned on and you move the cursor over your artwork, the cursor looks for objects, page boundaries, and intersections of construction guides to snap to that are within the tolerance range set in File > Document Properties > Snap tab.

You can use Guides in the following ways when you create, move, and transform objects.

  • When you create an object with the pen or shape tools, use the Guides to position the new object's anchor points relative to the other object.
  • When you move an object, use the Guides to align to the point on the object that you have selected. You can align to the anchor point at the corner of a selected object near the bounding box. To do so, select the object just inside the bounding box handle. If the tolerance is 5 pixels or greater, you can snap to the corner point from 5 pixels away.
  • When you transform an object, Guides appear to assist the transformation.
Note: When Snap to Grid is turned on, you cannot use Guides (even if the menu command is selected).

Drawing the Piece of Stationery

You can draw the diamond-shaped piece of stationery for the logo in a couple of ways. One way is to draw four-sided polygons (using the same methods you used to draw the triangle for the pencil tip). Another way is to draw using the Rectangle tool, and the Select and Transform tool.

Holding down <Shift> draws the rectangle from its center point rather than from the top left corner. Holding down both the <Ctrl> and <Shift> keys allows you to draw a square from its center point out.

Copying and Scaling Shapes

A final step in creating logos is to scale the artwork to a 1-inch square and make sure that the resized logo still presents a clear image. You'll use the bounding box feature in Inkscape to make a scaled copy of the logo.

  1. Double-click in the Zoom text box in the lower right-hand corner of the window to highlight its contents and type in 100 to zoom out to 100% magnification.

  2. Choose View > Page Grid to redisplay the grid.

  3. Choose Edit > Preferences > Behavior > Transforms. Select the Scale stroke width and Scale rounded corners in rectangles options. Leave the other settings as they are.

The Scale stroke width and Scale rounded corners in rectangles effects scale stroke weights and effects automatically. You can also choose these commands from the Select and Transform toolbar which appears when the Select and Transform tool is active ().

  1. Choose Edit > Select All to select all the objects in the logo, and then click the Select tool () in the toolbox.

  2. Move the Selection tool cursor into the artwork until the cursor changes into an arrow with an accompanying cross arrow ().

  3. Press the left mouse button and tap the spacebar once to make a copy of the artwork. Then drag the artwork copy to below the original and line up the left corner point on the logo with a grid line to make it easier to measure as you scale the copy.

  4. Using the Selection tool, select the bottom right corner point of the bounding box, hold down <Ctrl>, and drag the corner up and to the left to scale down the logo. Release the mouse button when the logo is about an inch wide.

    Holding down <Ctrl> as you drag the corner of the bounding box scales the objects proportionately.

  5. You can use various zoom options to zoom in on the smaller logo and check its clarity. Using the Selection tool, select the 1-inch-sized logo, then press the number 3 (Zoom to fit selection in window shortcut key).

  6. The spiral decoration needs to have a thicker stroke. Select the spiral with the Selection tool.

  7. Choose Object > Fill and Stroke. If the Stroke style tab isn't selected, then click on it.

  8. Choose px (pixels) from the Unit of Measurement drop-down menu. Change the width in the W text box to .9. Press <Enter> to set the new stroke measurement.

  9. Here is your finished product. Choose File > Save. Choose File > Close to close the file.

  2. How do you draw a square?
  3. How do you select the color fill of an object?
  4. How do you select the stroke color of an object?
  5. What is the difference between a view grid and a grid drawn with the grid extension tool?
  6. How do you rescale an object so that it scales up or down proportionately?

Review Answers

  1. The basic shape tools are the rectangle/square tool, ellipse/circle tool, polygon/star tool, and the spiral tool.
  2. Select the Rectangle/Square tool, press <Ctrl>, and drag to create the square.
  3. Select your object and, while the object is still selected, click on the desired color in the Color palette. The interior of the object is colored with that particular color. The stroke color is not affected.
  4. Select your object and, while the object is still selected, <Shift>-click on the desired color in the Color palette. The exterior boundary of the object is colored with that particular color. The fill color is not affected.
  5. The view grid acts as a guiding tool that you can use for precision placement of an object, including snapping. The view grid does not print with the artwork. A grid made with the grid extension tool does print.
  6. To rescale an object proportionately, press <Ctrl> and click-drag with the mouse to resize.
